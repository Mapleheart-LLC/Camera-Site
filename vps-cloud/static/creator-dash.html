Quick Guide

  1. Open OBS Studio (or any RTMP client).
  2. Go to Settings → Stream.
  3. Set Service to Custom…
  4. Paste the Server URL shown above.
  5. Paste the Stream Key for the camera you want to broadcast.
  6. Click Start Streaming — your viewers see you live immediately.

💡 To add or manage cameras, ask your administrator to use the admin panel. You can view the live status of all cameras here once you are streaming.
